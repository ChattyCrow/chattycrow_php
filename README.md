# ChattyCrow - PHP library

## Instalation

### Add ChattyCrow as a dependency to your *composer.json*
```json
{
  "repositories": [
    {
      "type": "vcs",
      "url":  "git@github.com:ChattyCrow/chattycrow_php.git"
    }
  ],
  "require": {
    "netbrick/chatty-crow": "*"
  }
}
```

### Install ChattyCrow
```sh
php composer.phar install
```
