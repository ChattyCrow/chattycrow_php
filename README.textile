h2. Instalation

h3. Add ChattyCrow as a dependency to your *composer.json*

<pre>
{
  "repositories": [
    {
      "type": "vcs",
      "url":  "git@github.com:ChattyCrow/chattycrow_php.git"
    }
  ],
  "require": {
    "netbrick/chatty-crow": "*"
  }
}
</pre>

h3. Install ChattyCrow

<pre>
php composer.phar install
</pre>
